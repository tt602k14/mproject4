# 🔐 THÔNG TIN ĐĂNG NHẬP ADMIN

## Tài khoản Admin mặc định

**Email:** `admin@htdgym.com`  
**Mật khẩu:** `admin123`

## Cách đăng nhập Admin

1. Mở ứng dụng HTD Gym
2. Tại màn hình đăng nhập, nhập:
   - Email: `admin@htdgym.com`
   - Mật khẩu: `admin123`
3. Nhấn "Đăng nhập"
4. Hệ thống sẽ tự động chuyển đến Admin Dashboard

## Chức năng Admin có sẵn

✅ **Admin Dashboard** - Tổng quan hệ thống  
✅ **Quản lý nội dung** - Thêm/sửa bài tập, video, workout  
✅ **Quản lý người dùng** - Xem, chỉnh sửa, khóa tài khoản  
✅ **Quản lý Premium** - Phân tích doanh thu, mã giảm giá  
✅ **Cài đặt hệ thống** - Database, bảo mật, backup  

## Lưu ý bảo mật

⚠️ **Quan trọng:** Sau khi đăng nhập lần đầu, hãy đổi mật khẩu admin trong phần "Cài đặt hệ thống > Cài đặt bảo mật"

## Khắc phục lỗi Database

Nếu gặp lỗi migration hoặc database:

1. Vào **Admin Dashboard > Cài đặt hệ thống**
2. Chọn **Database Settings > Reset database**  
3. Khởi động lại ứng dụng

Tài khoản admin sẽ được tự động tạo lại sau khi reset.
